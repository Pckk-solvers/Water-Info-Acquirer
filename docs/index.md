# Water-Info-Acquirer

水文データ取得・整理支援ツールです。公開されている水文データ／雨量データを取得し、Excel（必要に応じてCSV/Parquet）へ出力します。

## できること

- **国交省 水文データ取得**: 水位・流量・雨量を、指定期間で取得して Excel に出力します。
- **気象庁 雨量データ取得**: 都道府県・観測所を選び、指定期間の雨量データを取得して CSV/Excel に出力します。
