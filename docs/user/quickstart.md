# クイックスタート（exe）

## 1. ダウンロード

1. GitHub の Releases から最新版の `.zip` をダウンロードします。
2. ダウンロードした `.zip` を展開します。

!!! tip "展開先"
    展開先は、書き込み権限のあるフォルダ（例: `Documents` 配下）を推奨します。  
    出力フォルダ（例: `water_info/`, `jma/`）は、展開したフォルダ配下に作成されます。

## 2. 起動

1. 展開したフォルダ内の `.exe` をダブルクリックします。
2. 「ランチャー」画面が起動したら、使いたいアプリを選択します。

!!! note "初回起動が遅い場合"
    実行環境によっては画面が出るまで時間がかかる場合があります（数十秒程度）。  
    しばらく待っても起動しない場合は [トラブルシューティング](troubleshooting.md) を参照してください。

## 3. 取得して出力する

- 国交省の水文データ: [国交省 水文データ取得](water-info.md)
- 気象庁の雨量: [気象庁 雨量データ取得](jma-rainfall.md)
