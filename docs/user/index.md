# ユーザガイド（exe利用）

このセクションは、**配布されたexe（Windows）で使う方向け**に、画面の表示名そのままで操作手順をまとめます。

## まずやること

1. [クイックスタート（exe）](quickstart.md) でダウンロード〜起動まで実施
2. [ランチャー](launcher.md) で起動するアプリを選択

## 目的別

- **国交省の水位・流量・雨量をExcelにしたい**: [国交省 水文データ取得](water-info.md)
- **気象庁の雨量データをCSV/Excelにしたい**: [気象庁 雨量データ取得](jma-rainfall.md)
- **保存先やログを確認したい**: [出力先とログ](outputs-and-logs.md)
- **困ったとき**: [トラブルシューティング](troubleshooting.md)
