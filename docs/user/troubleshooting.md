# トラブルシューティング

## 起動しない／時間がかかる

- 初回起動は時間がかかる場合があります。しばらく待ってください。
- Windows の SmartScreen やウイルス対策でブロックされる場合があります。社内ルールに従って許可してください。

## 国交省 水文データ取得で出力されない

- 「指定期間に有効なデータが見つかりませんでした。」が出る  
  - 期間内にデータが無い可能性があります。短期間で試してから期間を延ばしてください。
- `[Error 13]` が出る  
  - 出力対象のExcelが開かれている可能性があります。Excelを閉じて再実行してください。

## 気象庁 雨量データ取得で取得できない

- 画面上部のステータスや `jma/logs/app.log` を確認してください。
- 都道府県一覧の「読み込み中…」が消えない  
  - 通信状況の影響が考えられます。時間を置く／再起動する／ネットワークを確認してください。

## 出力フォルダが作られない

- 展開先のフォルダに書き込み権限が無い可能性があります。`Documents` など書き込み可能な場所へ移動してください。
